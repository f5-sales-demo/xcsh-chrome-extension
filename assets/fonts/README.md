# Bundled font: MesloLGS NF

These four `.ttf` files are **MesloLGS NF** — the single-width ("S") Meslo LG variant
patched with [Nerd Fonts](https://github.com/ryanoasis/nerd-fonts) glyphs (powerline
separators + terminal icons). It is the font recommended by Powerlevel10k and used in the
maintainers' iTerm2 setup; bundling it makes the extension's chat window render identically
for every user regardless of what monospace fonts they have installed.

| File | Weight | Style |
| --- | --- | --- |
| `MesloLGS-NF-Regular.ttf` | 400 | normal |
| `MesloLGS-NF-Bold.ttf` | 700 | normal |
| `MesloLGS-NF-Italic.ttf` | 400 | italic |
| `MesloLGS-NF-Bold-Italic.ttf` | 700 | italic |
