生效日期：2026 年 7 月 3 日

**xcsh Chrome 扩展**（“本扩展”）是一款 Manifest V3 浏览器扩展，它让 xcsh AI 助手能够在您自己的浏览器中代表您操作 F5 Distributed Cloud 控制台。本政策说明本扩展访问哪些数据、如何使用这些数据以及本扩展绝不会做什么。

## 摘要

- 本扩展**仅**在 F5 Distributed Cloud 控制台页面（`*.volterra.us`、`*.console.ves.volterra.io`）上运行。
- 它读取这些页面的内容，以便助手能够理解并对屏幕上的内容采取操作。
- 这些内容和您的聊天消息**仅**发送到在您自己的计算机上本地运行的 xcsh 配套应用程序。
- 本扩展**不会**将您的数据发送给开发者，并且**不使用**任何分析、跟踪或广告。

## 本扩展访问哪些内容

当您位于 F5 Distributed Cloud 控制台页面上时（且仅在此时），本扩展可能会读取：

- **页面内容**——页面的结构和文本（无障碍树和 DOM）、当前 URL 以及表单字段值，以便助手能够定位和操作控制台控件。
- **您的聊天输入**——您在本扩展的侧边栏聊天中输入的消息。
- **诊断信号**——来自控制台页面的控制台日志消息和网络请求元数据，用于排查自动化问题。

本扩展不会读取任何其他网站上的页面。

## 如何使用您的数据

页面内容和您的聊天消息会通过本地通道（原生消息传递和 localhost 连接）发送到 **xcsh 配套应用程序**——一个您在自己计算机上安装并运行的命令行程序。配套应用程序运行 AI 助手，该助手决定采取哪些操作，并将指令发回给扩展，以便在控制台标签页中执行这些操作。

为了生成响应，配套应用程序会使用**您自己的**凭据，将您的请求和相关页面上下文发送给**您配置的** AI 模型提供商（例如 Anthropic API 或您组织的内部模型网关）。该处理受该提供商的条款和隐私政策约束。本扩展本身绝不会选择或联系模型提供商。

## 本扩展不会做什么

- 它**不会**将您的数据发送给本扩展的开发者或开发者运营的任何服务器。
- 它**不会**使用分析、遥测、跟踪、Cookie、指纹识别或广告。
- 它**不会**将您的数据出售或分享给第三方。
- 它**不会**为其单一目的（即应您的请求自动化 F5 Distributed Cloud 控制台）之外的任何目的收集数据。

## 本地存储

本扩展使用 Chrome 的扩展存储在您的浏览器中本地存储以下内容：

- 您的扩展设置。
- 您的聊天和会话历史记录，与其所属的控制台标签页相关联。

这些数据保留在您的设备上。您可以随时通过清除本扩展的数据或卸载本扩展来删除它们，卸载操作会删除其存储的数据。

## 数据保留

本扩展仅保留上述本地设置和历史记录，并且仅保留到您清除它们或卸载本扩展为止。开发者不保留任何内容，因为开发者不接收任何内容。

## 权限

本扩展仅请求代表您操作 F5 Distributed Cloud 控制台所需的权限：访问 F5 控制台域、读取和驱动控制台标签页的能力、用于设置和历史记录的本地存储，以及用于连接本地配套应用程序的原生消息传递。每项权限的理由都列在本扩展的 Chrome 网上应用店列表中。

## 本政策的变更

如果本政策发生变更，更新后的版本将在本页面上以新的生效日期发布。
