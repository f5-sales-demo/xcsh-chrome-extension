発効日: 2026年7月3日

**xcsh Chrome 拡張機能**（「本拡張機能」）は、xcsh AI アシスタントがお客様に代わってお客様自身のブラウザ内で F5 Distributed Cloud コンソールを操作できるようにする Manifest V3 ブラウザ拡張機能です。本ポリシーでは、本拡張機能がアクセスするデータ、そのデータの使用方法、および本拡張機能が決して行わないことについて説明します。

## 概要

- 本拡張機能は F5 Distributed Cloud コンソールのページ（`*.volterra.us`、`*.console.ves.volterra.io`）で**のみ**実行されます。
- それらのページからページコンテンツを読み取り、アシスタントが画面上の内容を理解して操作できるようにします。
- そのコンテンツとお客様のチャットメッセージは、お客様自身のコンピュータ上でローカルに実行される xcsh コンパニオンアプリケーションに**のみ**送信されます。
- 本拡張機能は開発者にお客様のデータを送信することは**なく**、分析、追跡、広告を一切**使用しません**。

## 本拡張機能がアクセスする内容

お客様が F5 Distributed Cloud コンソールのページにいる間、そしてそこでのみ、本拡張機能は以下を読み取る場合があります。

- **ページコンテンツ** — ページの構造とテキスト（アクセシビリティツリーおよび DOM）、現在の URL、フォームフィールドの値。これにより、アシスタントがコンソールのコントロールを特定して操作できます。
- **お客様のチャット入力** — 本拡張機能のサイドパネルチャットに入力するメッセージ。
- **診断シグナル** — コンソールページからのコンソールログメッセージとネットワークリクエストのメタデータ。自動化のトラブルシューティングに使用されます。

本拡張機能は、他のいかなる Web サイトのページも読み取りません。

## お客様のデータの使用方法

ページコンテンツとお客様のチャットメッセージは、**xcsh コンパニオンアプリケーション**（お客様自身のコンピュータにインストールして実行するコマンドラインプログラム）に、ローカルチャネル（ネイティブメッセージングと localhost 接続）経由で送信されます。コンパニオンアプリケーションは、実行するアクションを決定する AI アシスタントを実行し、コンソールタブでそれらを実行するための指示を本拡張機能に返します。

応答を生成するために、コンパニオンアプリケーションは、お客様の**設定した** AI モデルプロバイダー（例えば、Anthropic API またはお客様の組織の内部モデルゲートウェイ）に、**お客様自身の**認証情報を使用して、リクエストと関連するページコンテキストを送信します。その処理は、当該プロバイダーの利用規約およびプライバシーポリシーに準拠します。本拡張機能自体がモデルプロバイダーを選択したり接続したりすることは決してありません。

## 本拡張機能が行わないこと

- お客様のデータを本拡張機能の開発者、または開発者が運営するいかなるサーバーにも送信**しません**。
- 分析、テレメトリ、追跡、Cookie、フィンガープリンティング、または広告を使用**しません**。
- お客様のデータを第三者に販売または共有**しません**。
- 単一の目的（お客様の要求に応じて F5 Distributed Cloud コンソールを自動化すること）に関係のないいかなる目的のためにもデータを収集**しません**。

## ローカルストレージ

本拡張機能は、Chrome の拡張機能ストレージを使用して、以下をお客様のブラウザ内にローカルで保存します。

- お客様の拡張機能設定。
- お客様のチャットおよびセッション履歴（それが属するコンソールタブに関連付けられます）。

このデータはお客様のデバイス上に保持されます。本拡張機能のデータを消去するか、本拡張機能をアンインストールすることで、いつでもこれを削除できます。アンインストールすると、保存されたデータが削除されます。

## データ保持

本拡張機能は、上記のローカル設定と履歴のみを保持し、お客様がそれらを消去するか本拡張機能をアンインストールするまでの間のみ保持します。開発者は何も受け取らないため、開発者は何も保持しません。

## 権限

本拡張機能は、お客様に代わって F5 Distributed Cloud コンソールを操作するために必要な権限のみを要求します。すなわち、F5 コンソールドメインへのアクセス、コンソールタブを読み取り操作する機能、設定と履歴のためのローカルストレージ、およびローカルコンパニオンアプリケーションに到達するためのネイティブメッセージングです。各権限の正当な理由は、本拡張機能の Chrome ウェブストアの掲載ページに記載されています。
