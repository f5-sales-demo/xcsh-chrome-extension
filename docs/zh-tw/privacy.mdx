生效日期：2026 年 7 月 3 日

**xcsh Chrome 擴充功能**（下稱「本擴充功能」）是一款 Manifest V3 瀏覽器擴充功能，讓 xcsh AI 助理能夠在您自己的瀏覽器中，代表您操作 F5 Distributed Cloud 主控台。本政策說明本擴充功能所存取的資料、這些資料的用途，以及本擴充功能絕不會做的事。

## 摘要

- 本擴充功能**僅**在 F5 Distributed Cloud 主控台頁面（`*.volterra.us`、`*.console.ves.volterra.io`）上運作。
- 它會讀取這些頁面的內容，以便助理能夠理解並對螢幕上的內容採取行動。
- 這些內容以及您的聊天訊息**僅**傳送至在您自己電腦上本機運行的 xcsh 伴隨應用程式。
- 本擴充功能**不會**將您的資料傳送給開發者，也**不使用**任何分析、追蹤或廣告。

## 本擴充功能所存取的內容

當您位於 F5 Distributed Cloud 主控台頁面時，且僅在此情況下，本擴充功能可能會讀取：

- **頁面內容** — 頁面的結構與文字（協助工具樹和 DOM）、目前的 URL 以及表單欄位值，以便助理能夠定位並操作主控台控制項。
- **您的聊天輸入** — 您在本擴充功能側邊面板聊天中輸入的訊息。
- **診斷訊號** — 來自主控台頁面的主控台記錄訊息和網路要求中繼資料，用於排除自動化故障。

本擴充功能不會讀取任何其他網站上的頁面。

## 您的資料如何使用

頁面內容和您的聊天訊息會透過本機通道（原生訊息傳遞和 localhost 連線）傳送至 **xcsh 伴隨應用程式** — 一個您在自己電腦上安裝並運行的命令列程式。伴隨應用程式運行 AI 助理，由其決定要採取哪些行動，並將指令送回擴充功能，以在主控台分頁中執行這些行動。

為了產生回應，伴隨應用程式會使用**您自己的**憑證，將您的要求和相關的頁面內容傳送給**您所設定的** AI 模型供應商（例如 Anthropic API 或您組織的內部模型閘道）。該處理過程受該供應商的條款和隱私政策約束。本擴充功能本身絕不會選擇或聯絡任何模型供應商。

## 本擴充功能不會做的事

- 它**不會**將您的資料傳送給本擴充功能的開發者，或傳送給開發者所營運的任何伺服器。
- 它**不使用**分析、遙測、追蹤、Cookie、指紋辨識或廣告。
- 它**不會**出售或與第三方分享您的資料。
- 它**不會**為任何與其單一目的無關的用途收集資料：即應您的要求，自動化操作 F5 Distributed Cloud 主控台。

## 本機儲存

本擴充功能會使用 Chrome 的擴充功能儲存空間，將以下內容儲存在您的瀏覽器本機：

- 您的擴充功能設定。
- 您的聊天和工作階段歷史記錄，與其所屬的主控台分頁相關聯。

這些資料會保留在您的裝置上。您隨時可以透過清除本擴充功能的資料，或解除安裝本擴充功能（此舉會刪除其儲存的資料）來移除它們。

## 資料保留

本擴充功能僅保留上述本機設定和歷史記錄，且僅保留至您清除它們或解除安裝本擴充功能為止。開發者不保留任何內容，因為開發者不接收任何內容。

## 權限

本擴充功能僅要求代表您操作 F5 Distributed Cloud 主控台所需的權限：存取 F5 主控台網域、讀取並操控主控台分頁的能力、用於儲存設定和歷史記錄的本機儲存空間，以及用於連接本機伴隨應用程式的原生訊息傳遞。每項權限的理由都列在本擴充功能的 Chrome 線上應用程式商店清單中。
