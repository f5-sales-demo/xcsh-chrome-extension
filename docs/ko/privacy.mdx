발효일: 2026년 7월 3일

**xcsh Chrome 확장**("확장")은 xcsh AI 어시스턴트가 사용자를 대신하여 사용자 자신의 브라우저에서 F5 Distributed Cloud 콘솔을 조작할 수 있게 해주는 Manifest V3 브라우저 확장입니다. 본 정책은 확장이 어떤 데이터에 접근하는지, 그 데이터가 어떻게 사용되는지, 그리고 확장이 절대 하지 않는 것은 무엇인지를 설명합니다.

## 요약

- 확장은 F5 Distributed Cloud 콘솔 페이지(`*.volterra.us`, `*.console.ves.volterra.io`)에서 **만** 실행됩니다.
- 확장은 어시스턴트가 화면에 표시된 내용을 이해하고 조작할 수 있도록 해당 페이지의 콘텐츠를 읽습니다.
- 해당 콘텐츠와 사용자의 채팅 메시지는 사용자 자신의 컴퓨터에서 로컬로 실행되는 xcsh 컴패니언 애플리케이션으로 **만** 전송됩니다.
- 확장은 사용자의 데이터를 개발자에게 전송하지 **않으며**, 분석, 추적, 광고를 **전혀** 사용하지 않습니다.

## 확장이 접근하는 것

사용자가 F5 Distributed Cloud 콘솔 페이지에 있을 때, 그리고 그곳에서만, 확장은 다음을 읽을 수 있습니다:

- **페이지 콘텐츠** — 어시스턴트가 콘솔 컨트롤을 찾아 조작할 수 있도록 페이지의 구조와 텍스트(접근성 트리 및 DOM), 현재 URL, 폼 필드 값.
- **사용자의 채팅 입력** — 사용자가 확장의 사이드 패널 채팅에 입력한 메시지.
- **진단 신호** — 자동화 문제 해결에 사용되는 콘솔 페이지의 콘솔 로그 메시지 및 네트워크 요청 메타데이터.

확장은 다른 어떤 웹사이트의 페이지도 읽지 않습니다.

## 사용자의 데이터가 사용되는 방식

페이지 콘텐츠와 사용자의 채팅 메시지는 로컬 채널(네이티브 메시징 및 localhost 연결)을 통해 사용자가 자신의 컴퓨터에 설치하고 실행하는 명령줄 프로그램인 **xcsh 컴패니언 애플리케이션**으로 전송됩니다. 컴패니언 애플리케이션은 어떤 작업을 수행할지 결정하는 AI 어시스턴트를 실행하고, 콘솔 탭에서 이를 수행하도록 확장에 지시를 다시 보냅니다.

응답을 생성하기 위해, 컴패니언 애플리케이션은 사용자가 **구성한** AI 모델 제공자(예: Anthropic API 또는 조직의 내부 모델 게이트웨이)에게 사용자의 요청과 관련 페이지 컨텍스트를 **사용자 자신의** 자격 증명을 사용하여 전송합니다. 해당 처리는 그 제공자의 약관과 개인정보 보호정책의 적용을 받습니다. 확장 자체는 모델 제공자를 선택하거나 접촉하는 일이 결코 없습니다.

## 확장이 하지 않는 것

- 사용자의 데이터를 확장 개발자에게 또는 개발자가 운영하는 어떤 서버에도 전송하지 **않습니다**.
- 분석, 텔레메트리, 추적, 쿠키, 핑거프린팅, 광고를 사용하지 **않습니다**.
- 사용자의 데이터를 제3자에게 판매하거나 공유하지 **않습니다**.
- 사용자의 요청에 따라 F5 Distributed Cloud 콘솔을 자동화한다는 단일 목적과 무관한 어떤 목적으로도 데이터를 수집하지 **않습니다**.

## 로컬 저장소

확장은 Chrome의 확장 저장소를 사용하여 다음을 사용자의 브라우저에 로컬로 저장합니다:

- 사용자의 확장 설정.
- 해당 콘솔 탭과 연관된 사용자의 채팅 및 세션 기록.

이 데이터는 사용자의 기기에 남아 있습니다. 확장의 데이터를 지우거나, 저장된 데이터를 삭제하는 확장 제거를 통해 언제든지 이를 제거할 수 있습니다.

## 데이터 보존

확장은 위에서 설명한 로컬 설정 및 기록만을, 그리고 사용자가 이를 지우거나 확장을 제거할 때까지만 보존합니다. 개발자는 아무것도 수신하지 않기 때문에 아무것도 보존하지 않습니다.

## 권한

확장은 사용자를 대신하여 F5 Distributed Cloud 콘솔을 조작하는 데 필요한 권한만을 요청합니다: F5 콘솔 도메인에 대한 접근, 콘솔 탭을 읽고 구동하는 기능, 설정 및 기록을 위한 로컬 저장소, 그리고 로컬 컴패니언 애플리케이션에 도달하기 위한 네이티브 메시징. 각 권한의 정당성은 확장의 Chrome 웹 스토어 목록에 나열되어 있습니다.
