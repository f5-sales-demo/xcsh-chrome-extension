วันที่มีผลบังคับใช้: 3 กรกฎาคม 2026

**ส่วนขยาย Chrome ของ xcsh** ("ส่วนขยาย") เป็นส่วนขยายเบราว์เซอร์แบบ Manifest V3 ที่ให้ผู้ช่วย AI ของ xcsh ดำเนินการกับคอนโซล F5 Distributed Cloud แทนคุณ ภายในเบราว์เซอร์ของคุณเอง นโยบายนี้อธิบายว่าส่วนขยายเข้าถึงข้อมูลใดบ้าง ข้อมูลนั้นถูกใช้อย่างไร และส่วนขยายไม่เคยทำสิ่งใด

## สรุป

- ส่วนขยายทำงาน**เฉพาะ**บนหน้าคอนโซล F5 Distributed Cloud เท่านั้น (`*.volterra.us`, `*.console.ves.volterra.io`)
- อ่านเนื้อหาหน้าจากหน้าเหล่านั้นเพื่อให้ผู้ช่วยสามารถเข้าใจและดำเนินการกับสิ่งที่อยู่บนหน้าจอได้
- เนื้อหานั้นและข้อความแชทของคุณจะถูกส่งไปยัง**เฉพาะ**แอปพลิเคชันคู่หูของ xcsh ที่ทำงานอยู่ในเครื่องคอมพิวเตอร์ของคุณเองเท่านั้น
- ส่วนขยาย**ไม่**ส่งข้อมูลของคุณไปยังผู้พัฒนา และ**ไม่**ใช้การวิเคราะห์ การติดตาม หรือการโฆษณาใดๆ

## สิ่งที่ส่วนขยายเข้าถึง

ในขณะที่คุณอยู่บนหน้าคอนโซล F5 Distributed Cloud และเฉพาะที่นั่นเท่านั้น ส่วนขยายอาจอ่านสิ่งต่อไปนี้:

- **เนื้อหาหน้า** — โครงสร้างและข้อความของหน้า (accessibility tree และ DOM) URL ปัจจุบัน และค่าฟิลด์ฟอร์ม เพื่อให้ผู้ช่วยสามารถระบุตำแหน่งและดำเนินการกับตัวควบคุมของคอนโซลได้
- **ข้อความแชทของคุณ** — ข้อความที่คุณพิมพ์ในแชทของแผงด้านข้างของส่วนขยาย
- **สัญญาณการวินิจฉัย** — ข้อความบันทึกคอนโซลและข้อมูลเมตาของคำขอเครือข่ายจากหน้าคอนโซล ใช้เพื่อแก้ไขปัญหาการทำงานอัตโนมัติ

ส่วนขยายไม่อ่านหน้าเว็บบนเว็บไซต์อื่นใด

## วิธีที่ข้อมูลของคุณถูกใช้

เนื้อหาหน้าและข้อความแชทของคุณจะถูกส่งไปยัง**แอปพลิเคชันคู่หูของ xcsh** — โปรแกรมบรรทัดคำสั่งที่คุณติดตั้งและเรียกใช้บนเครื่องคอมพิวเตอร์ของคุณเอง — ผ่านช่องทางในเครื่อง (native messaging และการเชื่อมต่อ localhost) แอปพลิเคชันคู่หูจะเรียกใช้ผู้ช่วย AI ที่ตัดสินใจว่าจะดำเนินการใด และส่งคำสั่งกลับไปยังส่วนขยายเพื่อดำเนินการในแท็บคอนโซล

เพื่อสร้างการตอบกลับ แอปพลิเคชันคู่หูจะส่งคำขอของคุณและบริบทหน้าที่เกี่ยวข้องไปยังผู้ให้บริการโมเดล AI **ที่คุณกำหนดค่า** (ตัวอย่างเช่น Anthropic API หรือเกตเวย์โมเดลภายในองค์กรของคุณ) โดยใช้ข้อมูลรับรอง**ของคุณเอง** การประมวลผลนั้นอยู่ภายใต้ข้อกำหนดและนโยบายความเป็นส่วนตัวของผู้ให้บริการนั้น ตัวส่วนขยายเองไม่เคยเลือกหรือติดต่อผู้ให้บริการโมเดล

## สิ่งที่ส่วนขยายไม่ทำ

- **ไม่**ส่งข้อมูลของคุณไปยังผู้พัฒนาส่วนขยายหรือเซิร์ฟเวอร์ใดๆ ที่ดำเนินการโดยผู้พัฒนา
- **ไม่**ใช้การวิเคราะห์ เทเลเมทรี การติดตาม คุกกี้ การเก็บลายนิ้วมือ (fingerprinting) หรือการโฆษณา
- **ไม่**ขายหรือแชร์ข้อมูลของคุณกับบุคคลที่สาม
- **ไม่**เก็บรวบรวมข้อมูลเพื่อวัตถุประสงค์ใดๆ ที่ไม่เกี่ยวข้องกับวัตถุประสงค์เดียวของมัน: การทำงานอัตโนมัติของคอนโซล F5 Distributed Cloud ตามคำขอของคุณ

## พื้นที่จัดเก็บในเครื่อง

ส่วนขยายจัดเก็บสิ่งต่อไปนี้ในเครื่องภายในเบราว์เซอร์ของคุณ โดยใช้พื้นที่จัดเก็บส่วนขยายของ Chrome:

- การตั้งค่าส่วนขยายของคุณ
- ประวัติแชทและเซสชันของคุณ ที่เชื่อมโยงกับแท็บคอนโซลที่มันเป็นของ

ข้อมูลนี้จะอยู่บนอุปกรณ์ของคุณ คุณสามารถลบออกได้ทุกเมื่อโดยการล้างข้อมูลของส่วนขยายหรือถอนการติดตั้งส่วนขยาย ซึ่งจะลบข้อมูลที่จัดเก็บไว้

## การเก็บรักษาข้อมูล

ส่วนขยายเก็บรักษาเฉพาะการตั้งค่าและประวัติในเครื่องตามที่อธิบายไว้ข้างต้นเท่านั้น และเฉพาะจนกว่าคุณจะล้างข้อมูลเหล่านั้นหรือถอนการติดตั้งส่วนขยาย ผู้พัฒนาไม่เก็บรักษาสิ่งใด เพราะผู้พัฒนาไม่ได้รับสิ่งใด

## สิทธิ์การอนุญาต

ส่วนขยายร้องขอเฉพาะสิทธิ์ที่จำเป็นในการดำเนินการกับคอนโซล F5 Distributed Cloud แทนคุณ: การเข้าถึงโดเมนคอนโซล F5 ความสามารถในการอ่านและควบคุมแท็บคอนโซล พื้นที่จัดเก็บในเครื่องสำหรับการตั้งค่าและประวัติ และ native messaging เพื่อเข้าถึงแอปพลิเคชันคู่หูในเครื่อง เหตุผลของแต่ละสิทธิ์ได้ระบุไว้ในรายการของส่วนขยายบน Chrome Web Store

## การเปลี่ยนแปลงนโยบายนี้

หากนโยบายนี้มีการเปลี่ยนแปลง เวอร์ชันที่ปรับปรุงแล้วจะถูกเผยแพร่บนหน้านี้พร้อมวันที่มีผลบังคับใช้ใหม่
